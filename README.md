# Personal Portfolio Website

A modern and responsive portfolio website built using Next.js and Tailwind CSS to showcase my projects, technical skills, and experience in Artificial Intelligence, Machine Learning, Data Science, and Software Development.

## Features

* Responsive design
* Modern UI/UX
* Project showcase section
* Skills and expertise section
* About me section
* Contact links
* Fast and optimized performance

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS

## Purpose

This portfolio serves as a central hub for recruiters, collaborators, and employers to explore my work, technical capabilities, and professional background.
